Minesweeper in Love2d
======
Minesweeper written in Lua for [Love2d](https://love2d.org/)! 
<br/>

### Features
`left click` to reveal tile<br/>
`right click` to flag tile<br/>
`middle click` to mark tile as "unknown"<br/>
`space` to reset grid on win/lose<br/>
